💸 HisabKitab – Expense Tracker & Trip Splitter 💸

📌 Overview: HisabKitab is a web-based application designed to help users manage their daily expenses and split group expenses during trips or shared activities. It provides a simple and efficient way to track spending and calculate fair settlements among multiple users.

🚀 Features:
1. User Signup and Login system
2. Daily Expense Tracking
3. Create and manage trips
4. Add multiple members in a trip
5. Record shared expenses
6. Automatic expense splitting and settlement calculation
7. View trip summary and balances
8. Delete expenses and trips

🛠️ Technologies Used:
Frontend: HTML, CSS (Jinja Templates) Backend: Flask (Python) Database: MongoDB Atlas Deployment: Render

⚙️ How to Use: (a) Sign up and log in to your account (b) Add your daily expenses (c) Create a trip and add members (d) Record shared expenses during the trip (e) View the summary to see who owes whom (f) Manage or delete expenses as needed

🌐 Live Website 
👉 https://hisab-kitab-hezo.onrender.com [Backend]
👉 https://hisab-kitab-alpha.vercel.app [Frontend]

🎯 Purpose: This project is developed to solve real-world problems of managing personal and group expenses. It eliminates manual calculations, reduces confusion, and ensures fair distribution of costs.

🔒 Note: Sensitive information like database credentials is stored using environment variables for security.

📌 Future Scope: Payment integration (UPI/QR settlement) Graphs and expense analytics Notifications and reminders Mobile app version Export reports (PDF/Excel)
